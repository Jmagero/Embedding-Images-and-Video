Navigation bar It has an embedded image that represents a logo. The logo is aligned to the leftmost part of the navbar. It has an input tag that represents a search bar. The search bar is aligned to the middle of the navbar. It has a button tag that represents the search button. The search button is at the end (rightmost part) of the search bar. It has at least three embedded images on the rightmost part of the navbar that represent icons and/or an avatar.

Code Review Request: This is the form a team has to submit to get a TSE to review their project. Any member of the team can submit the request. A project without a review from a TSE doesn’t count towards progression in the program. If the TSE decides the project requires changes, the team has to do them and submit a new Code Review Request. Otherwise, the team has to submit a Project Completion Form Project Completion Form: This is the form you fill out after a TSE says your project is complete and no further corrections need to be made. This is your chance to give us feedback about the project. Did you like it? How useful do you find it? These are personal opinions so they should be submitted individually.
